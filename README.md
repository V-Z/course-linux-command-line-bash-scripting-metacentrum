# Kurz práce v příkazové řádce Linuxu nejen pro MetaCentrum (MB120C23)

Nebojte se příkazové řádky Linuxu! Je to mocný a přívětivý nástroj umožňující efektivně zpracovat i velká data a zautomatizovat činnosti. Prakticky shodně funguje příkazová řádka i v Apple macOS, BSD a dalších UNIXových systémech, nejen v Linuxu. Kurz je vhodný pro úplné začátečníky i mírně pokročilé. Jediným vstupním požadavkem je zájem (nebo potřeba) pracovat v příkazové řádce, typicky na linuxovém výpočetním serveru. Kurz bude probíhat v Linuxu, ale většina věcí funguje stejně na macOS a dalších UNIXových systémech.

Znalosti práce v Linuxu/UNIXu se hodí např. při zpracování rozsáhlých molekulárních a jiných dat. MetaCentrum je služba CESNETu poskytující přístup k obrovské výpočetní kapacitě, kterou mohou všichni studenti a zaměstnanci akademických institucí v Česku využít pro své výpočty. Znalosti z kurzu tam ihned můžete zúročit. Kurz poskytuje teoretický úvod i řadu praktických příkladů a cvičení.

Kurz poběží turnusově 3 dny. Konzultace a zápočty jsou možné kdykoliv po předchozí domluvě.

Bude-li se kurzu účastnit alespoň jeden člověk nemluvící česky, kurz bude anglicky.

# Course of work in Linux command line not only for MetaCentrum (MB120C23)

Don't be afraid of the command line! It is a user-friendly, powerful tool that allows you to process large amounts of data and automate tasks. The command line is practically identical in Apple macOS, BSD, and other UNIX-based systems, not just Linux. This course is designed for total beginners as well as intermediate and advanced students. The only requirement is an interest in or need to work with the command line, typically on a Linux computing server.

The course will be taught in Linux, but most of the points are also applicable to other UNIX systems, such as macOS.

Knowledge of Linux/UNIX is useful for working with large molecular data, among other things. MetaCentrum is a CESNET service that provides access to significant computational capacity for all students and employees of Czech academic institutions for processing their data. You can immediately apply the skills you learn in this course. The course includes a theoretical introduction, as well as plenty of practical examples and exercises.

The course will be taught over three days. Consultations and exams can be scheduled at any time with prior agreement.
